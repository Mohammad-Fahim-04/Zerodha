# Zerodha

A full-stack stock trading platform inspired by Zerodha, built using React, Node.js, Express, and MongoDB.

## Features
- Landing Page
- Trading Dashboard
- Holdings Management
- Positions Tracking
- Orders Module
- Responsive Design
- Unit Testing with Jest

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- Bootstrap
- Jest & React Testing Library
